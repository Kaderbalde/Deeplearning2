# 🧠 Deep Learning - Prédiction du mot suivant

Ce dépôt contient un script Python utilisant un modèle **LSTM** pour prédire le **mot suivant** dans une phrase en français, à partir d’un petit corpus d'exemples. Le modèle est entraîné avec **Keras** et **TensorFlow**.

## 📄 Fichier principal

- `predict_next_word.py` : script principal qui entraîne un modèle LSTM pour la modélisation du langage et permet d’interagir avec l’utilisateur en prédiction du mot suivant.

## 🚀 Fonctionnalités

- Traitement automatique d’un corpus de phrases simples
- Tokenisation et vectorisation des séquences
- Entraînement d’un modèle LSTM
- Interface interactive en ligne de commande pour tester des prédictions
